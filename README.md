# 💰 FinanceFlex

**FinanceFlex** é um aplicativo de gestão financeira pessoal moderno, desenvolvido com **React Native** e **Expo**. Ele foi projetado para ajudar usuários a terem controle total sobre suas finanças, permitindo o gerenciamento de contas, transações, cartões de crédito e metas de economia (cofrinhos) de forma intuitiva e eficiente.

## 🚀 Funcionalidades Principais

### 🏦 Gestão de Contas
- Cadastro e edição de múltiplas contas (Banco, Dinheiro, Investimentos).
- Visualização de saldo individual e saldo total consolidado.
- Sistema inteligente de exibição: exibe as 3 contas principais com opção de "Ver todas".

### 💸 Controle de Transações
- Registro detalhado de Receitas, Despesas e Transferências entre contas.
- Categorização inteligente para melhor organização dos gastos.
- Histórico de transações recentes acessível diretamente na dashboard.

### 💳 Cartões de Crédito
- Gerenciamento completo de cartões com controle de limite total e disponível.
- Acompanhamento de faturas e datas de fechamento/vencimento.
- Registro de compras parceladas com geração automática de faturas futuras.

### 🐷 Cofrinhos (Metas de Economia)
- Criação de metas personalizadas com valor alvo e cor de identificação.
- **Depósitos e Retiradas**: Sistema integrado que retira ou devolve o saldo para as suas contas reais.
- Acompanhamento de progresso visual com barra de progresso e porcentagem em tempo real.

### 📅 Planejamento Financeiro
- Calendário de contas do mês para visualização de compromissos financeiros.
- Resumo mensal automático com taxa de economia e balanço entre receitas e despesas.

### 🔐 Segurança e Sincronização
- **Privacidade**: Opção de ocultar valores sensíveis na tela inicial.
- **Sincronização Cloud**: Integração com Firebase para manter seus dados seguros e acessíveis em múltiplos dispositivos.
- **Backup Local**: Sistema de exportação e importação de dados para segurança extra.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza as tecnologias mais modernas do ecossistema JavaScript/TypeScript:

- **[React Native](https://reactnative.dev/)** & **[Expo](https://expo.dev/)** - Desenvolvimento mobile cross-platform.
- **[TypeScript](https://www.typescriptlang.org/)** - Tipagem estática para maior robustez do código.
- **[Firebase](https://firebase.google.com/)** - Autenticação e banco de dados em tempo real.
- **[AsyncStorage](https://react-native-async-storage.github.io/async-storage/)** - Persistência de dados local.
- **[Expo Router](https://docs.expo.dev/router/introduction/)** - Navegação baseada em arquivos.
- **[Vector Icons](https://icons.expo.fyi/)** - Ícones intuitivos para melhor UX.

## ⚙️ Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/kaiquebazil/FinanceFlexReact.git
   ```

2. **Instale as dependências:**
   ```bash
   yarn install
   # ou
   npm install
   ```

3. **Inicie o projeto:**
   ```bash
   npx expo start
   ```

4. **Abra o aplicativo:**
   - Use o aplicativo **Expo Go** no seu celular para ler o QR Code.
   - Ou pressione `w` para abrir na versão web.
