# FinanceFlexReact

Aplicativo móvel de gestão financeira pessoal construído com React Native.

## 🚀 Funcionalidades

- Controle de transações e orçamentos
- Gerenciamento de cartões de crédito
- Sincronização com Firebase
- Projeções de economia (Piggy Bank)
- Backup e restauração de dados

## 🛠️ Tecnologias Utilizadas

- React Native
- Expo
- TypeScript
- Firebase
- Context API

## 📦 Como Executar

1. Clone o repositório
2. Siga as instruções específicas do ambiente.
